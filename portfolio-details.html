Gundam Fighter - C# Dungeon Application

Project information

  • Category: C# Console
  • School: Centriq
  • Project date: 03, March 2023
  • Project URL: https://github.com/SantiagoEfrain/GundamSolution

This project was designed to be a simple dungeon style application. I spent many hours finding ways to implement the style of gameplay that I wanted to convey. Using what methods I was instructed based off our class dungeon app, I found how to make my own app with its own rules of play. I did so using a source of content that I love, Gundam. With that I used various C# techniques to create an interactive UI with a PIN accessed game. Using loops you can defeat enemies with three different types of weapons and level up your given mobile suits to stronger ones making the game easier to progress through. At the end you can exit the loop and your total score will be submitted to you. I learned a lot from this project as I took many hours learning which functions to use and how to make the game operate as I intended. Instead of looking cookie cutter to the class code I wanted to implement different rules.
